AURA VISUAL UPGRADE AUTO-INSTALLER
==================================

Version 0.3.0 by Srixun
Community: https://Discord.gg/AuraPub - PvP'ers welcome.

Run "START HERE - Install AURA Visual Upgrade.cmd".

The installer reads the official Ascension Launcher's cached installation
path and checks common locations on every drive. If multiple clients exist, it
shows a numbered selection. Manual folder selection remains available.

The embedded addon archive is SHA-256 verified before any existing addon is
changed. Updates preserve the previous addon folder with a timestamp, install
through a temporary directory, and verify every installed file.

Every install checks https://github.com/Srixun/AURA-Visual-Upgrade/releases for
a newer version. GitHub downloads are accepted only when they match the
release's SHA-256 asset. If GitHub is unavailable during a normal install, the
verified bundled version is used. Run "Check for AURA Updates.cmd" to require
an online update check.

Close Ascension before installing. After installation, launch the game and use
/auravis, the AV minimap button, or Interface Options.

This package installs only the WoW addon. DX12, ReShade, AURA Visual Sync, and
frame-generation support are available in the separate full graphics companion
package.

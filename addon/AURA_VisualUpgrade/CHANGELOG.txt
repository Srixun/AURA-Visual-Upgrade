AURA VISUAL UPGRADE CHANGELOG
=============================

0.4.0 - 2026-07-18
------------------
- Added hardware-neutral Adaptive Optimization with selectable targets and behavior goals.
- Added a 15-second context-aware benchmark and profile recommendation.
- Added verified gradual reductions, benefit checks, cooldowns, conservative recovery, and retained baseline restoration.
- Added world-transition warm-up, frame/VSync ceiling awareness, background-throttle filtering, and context safety floors.
- Kept encounter particles, projected textures, view distance, frame caps, texture quality, restart-required controls, and external graphics outside automatic control.
- Removed fixed hardware, frame-cap, renderer, and ReShade assumptions from game profiles.
- Added a draggable minimap icon with a persisted position.
- Rebuilt the dashboard with native WotLK graphics and controls.
- Added modular Lua themes, ordered skin providers, and optional LibSharedMedia support.
- Made companion external sync opt-in and preserved existing frame-cap and VRAM values in ordinary external profiles.
- Fixed MSAA changes being reported as failed while Ascension was waiting for a graphics restart to update the live CVar.
- Replaced the ReShade approval confirmation with an explicit unrestricted-depth option and a policy/anti-cheat risk explanation.

0.3.0 - 2026-07-18
------------------
- Added peer-reported version awareness over the native addon-message channel.
- Added an update badge, manual peer check, and selectable official GitHub Releases link.
- Added the Raid profile for encounter clarity and stable crowded-area performance.
- Added supported Ascension controls for raid particles, extended shadows, model textures, background FPS, and other safe graphics CVars.
- Unsupported CVar controls are now hidden instead of being staged or reported as failures.
- Kept gxFixLag as an explicit troubleshooting control that profiles never change.

0.2.0 - 2026-07-18
------------------
- Updated author metadata to Srixun.
- Added dynamic real-FPS and Smooth Motion displayed-FPS targets.
- Added an About button to the main dashboard.
- Expanded the Interface Options About page and configuration launcher.
- Added complete /auravis command guidance.
- Added the supplied circular AURA minimap artwork.
- Tightened the wordmark crop and removed the overlapping Blizzard border so the logo remains readable at minimap size.
- Added the AURA community Discord and PvP welcome message to About.

0.1.0 - 2026-07-18
------------------
- Initial public-preview release.
- Added native in-game graphics dashboard and /auravis.
- Added Performance, Balanced, Quality, and Recommended profiles.
- Added staged apply flow and graphics-restart confirmation.
- Added external DX11/DX12, ReShade, effect, and Smooth Motion requests.
- Added SavedVariables bridge for AURA Visual Sync.
